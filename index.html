Projects:

Mixed Messages

The aim of this project was to generate a random phrase. Working with 2 fellow students we came up with the concept of a animal doing something to another animal. As the 2 other guys had done English as a foreign language courses one suggested we also include a phrase showing the same action in the past tense.

I went for a simple style for this project as it was to showcase the coding. I tried for a businesslike plain style for the styling.

Jammming

My first project in React and interacting with APIs, this downloads a track listing based on a search term from Spotify and allows you to build a playlist. It is a demonstration of passing state through the react component structure using props and callbacks. This was the first time i used authentication to a third party site.

I decided to style it like an old fashioned led matrix car stereo interface.
